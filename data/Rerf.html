Task

Build a Random Forest Enhanced Regression model from the data in "random_forest_enhanced_model.csv", train it, evaluate its performance, and use it to make predictions.

The core idea of RERF is:

Decompose the prediction into a parametric component (global trend) + a nonparametric component (local correction via RF).

Mathematically:

^ ( )

^ Parametric + ( ) Nonparametricresidualcorrection Y ^ (X)= Parametric X ^

+ Nonparametricresidualcorrection RF(X)

from sklearn.metrics import r2_score
import numpy as np
import pandas as pd

# Prepare actual and prediction columns
df_metrics = pd.DataFrame({
    'Actual': gdf['hpm'],
    'Linear Regression': gdf['Linear Regression'],
    'Random Forest': gdf['Random Forest'],
    'ReRF': gdf['ReRF']
})

# Function to compute all requested metrics
def compute_metrics(y_true, y_pred):
    eps = 1e-9  # to avoid division by zero
    pe = (y_true - y_pred) / (y_true + eps)
    ape = np.abs(pe)
    fsd = np.std(ape)
    pe10 = (ape < 0.10).mean()
    rt20 = (ape > 0.20).mean()
    r2 = r2_score(y_true, y_pred)
    return {
        'R': round(r2, 4),
        'APE Mean': round(np.mean(ape), 4),
        'FSD': round(fsd, 4),
        'PE10 (%)': round(pe10 * 100, 2),
        'RT20 (%)': round(rt20 * 100, 2)
    }

# Apply to each model
metrics_dict = {}
for model in ['Linear Regression', 'Random Forest', 'ReRF']:
    metrics_dict[model] = compute_metrics(df_metrics['Actual'], df_metrics[model])

# Create a summary DataFrame
metrics_df = pd.DataFrame(metrics_dict).T
metrics_df.index.name = 'Model'

print(metrics_df)
